# 第7章-初始化程序

## 总结

### main.c程序

* 初始化阶段
  * 首先如何分配使用系统物理内存
  * 调用内核各个部分的初始化函数分别对内存管理、中断处理、块设备、字符设备、进程管理以及硬盘和软盘等硬件进行初始化
  * 在完成这些初始化后，系统各部分以及处于可运行状态。
  * 程序手动移到进程0运行
  * 进程0调用fork()创建init进程

![1528847185838.png](image/1528847185838.png)

* init功能分为四个部分
  * 安装根文件系统
  * 显示系统信息
  * 运行系统初始资源配置rc中的命令
  * 执行登录shell程序
